# End-Colonialism
This could be my final mod before the ultimate mod pack, because I'm running out of ideas. The End is long overdue for an update. However, instead of making it another nether update as most mods do, I'm picturing it as a more colonial update. The end should have a different identity other than "just another space dimension", that's in The Cold Above.
